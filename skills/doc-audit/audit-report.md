---

## 각 Task 결과 반환 형식

각 Task(deps-audit, pattern-audit, rulecheck-audit)는 아래 항목을 포함하여 결과를 반환한다:

- **대상 문서 + 섹션**
- **현재 내용** / **제안 내용**
- **근거** (의존성 파일, git diff, 코드 위치 등)
- **중요도** (Critical / Major / Minor)
