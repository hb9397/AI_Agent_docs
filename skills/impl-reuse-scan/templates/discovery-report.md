## 빈 결과 처리

🔴/🟡 후보가 모두 없으면 다음 형식으로 출력한다.

```markdown
# 공통 자산 발견 리포트

**스캔 대상**: {Phase N} / {태스크 ID 목록}

스캔 결과 재사용 후보가 발견되지 않았습니다.
신규 자산 생성으로 진행하셔도 충돌 위험이 낮습니다.

(참고용으로 발견된 무관 자산: {N}건)
```

## 스캔 범위/한계 명시

리포트 하단에 항상 다음 문구를 포함한다.

```markdown
---

## 스캔 범위

- 스캔한 디렉토리: {경로 목록}
- 스캔하지 않은 영역: {제외 경로 또는 "없음"}
- 카테고리: {BE / FE / 공통 등}

이 결과는 표면 패턴 매칭(grep/glob) 기반입니다.
AST 분석이 아니므로 일부 변형(동적 import, 메타프로그래밍)은 누락될 수 있습니다.
중요한 자산은 직접 코드베이스에서 한 번 더 확인을 권장합니다.
```
