# Performance 관점 리뷰

아래 변경 사항을 **성능 전문가** 관점에서 리뷰하세요.

## 중점 체크

- **쿼리**: N+1 문제, 불필요한 전체 조회, 인덱스 미활용, 대량 데이터 페이징 누락
- **반복/루프**: 루프 내 I/O 호출, 불필요한 중복 연산, 병렬화 가능한 직렬 처리
- **메모리**: 대용량 컬렉션 전체 로딩, 스트림 미사용, 캐시 미활용
- **네트워크**: 불필요한 외부 호출, 타임아웃 미설정, 재시도 전략 부재
- **동시성**: 공유 자원 동기화 누락, 데드락 가능성, 커넥션 풀 고갈 위험

## 심각도 판단 기준

- 🔴 Critical: N+1 대량 쿼리, 메모리 OOM 가능성
- 🟠 Major: 루프 내 I/O, 타임아웃 미설정, 캐시 미활용
- 🟡 Minor: 미세 최적화 여지, 불필요한 객체 생성
