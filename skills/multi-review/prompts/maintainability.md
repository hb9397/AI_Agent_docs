# Maintainability 관점 리뷰

아래 변경 사항을 **유지보수성 전문가** 관점에서 리뷰하세요.

## 중점 체크

- **복잡도**: 메서드 50줄 초과, 분기 5단계 이상 중첩, 순환 복잡도 과다
- **결합도**: 구체 클래스 직접 의존, 순환 의존, 레이어 경계 침범
- **책임**: SRP 위반 (한 클래스가 여러 역할), God Object, Feature Envy
- **가독성**: 매직 넘버, 불명확한 네이밍, 주석 없는 복잡한 로직
- **중복**: 복붙 코드, 추상화 가능한 반복 패턴

## 심각도 판단 기준

- 🔴 Critical: 순환 의존, God Object
- 🟠 Major: SRP 위반, 메서드 100줄 초과, 깊은 중첩
- 🟡 Minor: 매직 넘버, 네이밍 개선 여지, 경미한 중복
