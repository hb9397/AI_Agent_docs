# Security 관점 리뷰

아래 변경 사항을 **보안 전문가** 관점에서 리뷰하세요.

## 중점 체크

- **인젝션**: SQL, NoSQL, OS Command, LDAP 인젝션 가능 지점
- **인증/인가**: 인증 우회, 권한 검증 누락, 수평/수직 권한 상승
- **민감 정보**: 하드코딩된 시크릿, 로그에 민감 정보 노출, 응답에 불필요한 데이터 포함
- **입력 검증**: 사용자 입력 미검증, XSS 가능 지점, 파일 업로드 검증 누락
- **암호화**: 평문 저장/전송, 취약한 해시 알고리즘 (MD5, SHA1)

## 심각도 판단 기준

- 🔴 Critical: 인젝션, 인증 우회, 민감 정보 평문 노출
- 🟠 Major: 권한 검증 누락, 입력 미검증
- 🟡 Minor: 불필요한 정보 응답 포함, 로깅 과다
