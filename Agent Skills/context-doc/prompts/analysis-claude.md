# CLAUDE.md 추출 기준 (analysis-claude)

설계 문서에서 CLAUDE.md 생성에 필요한 항목만 추출한다.

---

## 추출 항목

### 1. 프로젝트 맥락
- 프로젝트명 및 부제 (한 줄 설명)
- 무엇을 하는 시스템인가 (도메인, 목적)
- 어떤 사용자가 쓰는가
- FE / BE / AI 역할 분리 구조

### 2. 기술 스택
- 레이어별 라이브러리 + 버전
- 패키지 매니저
- 실행 환경 (로컬 / 컨테이너 / 클라우드)

### 3. 아키텍처 & 디렉토리 구조
- 폴더 트리 (역할이 있는 파일만, 전체 나열 금지)
- 페이지/화면 목록 및 각 역할
- 핵심 컴포넌트 목록 및 역할

### 4. 백엔드 통신 규칙
- API 엔드포인트 전체 목록 (메서드, 경로, 설명)
- WebSocket 엔드포인트 및 메시지 타입
- Base URL, 포트 구성
- API 집중 관리 방식 (예: client.js 패턴)

### 5. 환경변수
- 개발/배포 환경 분리 여부
- 환경변수 목록 (변수명, 설명, 예시값)
- 환경별 동작 차이 (예: noVNC 분기)

### 6. 핵심 도메인 개념
- 프로젝트 고유 용어 및 정의
- 핵심 식별자 (예: site_name, user_id)

### 7. 실행 방법
- 개발 환경 실행 명령
- 배포 환경 실행 명령
- 초기 세팅 명령 (최초 1회)
