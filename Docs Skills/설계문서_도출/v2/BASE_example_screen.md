---

## 1. 한 줄 요약

```
사용자가 예약 사이트의 UI 요소를 클릭하면 AI가 역할을 추론하고 저장하는 온보딩 화면
```

---

## 2. 범위

**포함**
- 가상 화면(:6080) 임베드
- 클릭한 요소 목록 표시 + AI 추론 결과 확인
- 저장

**제외**
- 로그인
- 사이트 URL 등록 (이전 화면에서 완료)

---

## 3. 사용자와 맥락

```
최초 사이트 등록 시 1회만 진행.
클릭 → 확인 → 저장의 반복. 3~5분 안에 끝나야 함.
```

---

## 4. 핵심 흐름

```
1. 화면 왼쪽: 가상 브라우저(noVNC)에서 예약 사이트 열림
2. 사용자가 요소 클릭 (예: 출발역 입력칸)
3. 화면 오른쪽: 클릭된 요소 + AI 추론 역할명 표시
4. 사용자가 역할명 확인 (수정 가능)
5. 전체 요소 확인 완료 → 저장 버튼
```

---

## 5. 데이터

| 이름 | 설명 | 비고 |
|------|------|------|
| element.selector | 클릭된 DOM 요소 식별자 | xpath or css |
| element.role | AI가 추론한 역할명 | "출발역", "날짜" 등 |
| element.confirmed | 사용자 확인 여부 | boolean |

---

## 6. 결정된 것 / 결정 안 된 것

| 결정된 것 | 결정 안 된 것 |
|-----------|---------------|
| 클릭 감지는 noVNC 이벤트로 | AI 추론 실패 시 fallback UI |
| 좌우 분할 레이아웃 | 요소 클릭 취소/수정 UX |

---

## 7. 제약

```
- 가상화면은 :6080 포트로 iframe 임베드
- AI 추론 응답은 2초 이내 (느리면 스켈레톤 표시)
```
